## Features

- Fixed Navigation Bar: Sticky navbar that remains visible on scroll
- Light & Dark Mode: Toggle between light and dark themes using JavaScript
- Responsive Design: Optimized for desktop, tablet, and mobile screens
- Micro-Interactions: Smooth hover effects on buttons and cards
- Framework-Free: Built without Bootstrap, Tailwind, or external UI libraries

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** CSS Flexbox, Grid, Transitions, CSS Variables
- **Interactivity:** DOM manipulation using JavaScript
